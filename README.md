## Usage

Install via npm

```sh
npm i kanvas-dialog
```

Load from CDN

```js
import "https://cdn.jsdelivr.net/npm/kanvas-dialog@latest";
```

[Example](https://raw.githubusercontent.com/hata6502/kanvas-landing-page/main/content/_index.md)

## Development

```bash
npm run build -- --mode development && npm run serve
```
